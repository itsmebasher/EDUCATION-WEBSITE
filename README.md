# EDUCATION-WEBSITE

A starter education website for SDG 4: Quality Education. Includes study materials, interactive quizzes, video lessons, and a forum placeholder.

## Run locally

1. Open `index.html` in your browser.
2. Or run a local server:
   - `python3 -m http.server 8000`
   - Open `http://localhost:8000`

## Project layout

- `index.html`: main page layout
- `styles.css`: theme and responsive styles
- `script.js`: interactive quiz logic

## Goals

- Offer an accessible learning hub
- Support quick study and self-assessment
- Provide a template for adding resources and forums

The education website is designed to support SDG 4: Quality Education by providing students with easy access to learning resources anytime and anywhere. The platform offers study materials, interactive quizzes, video lessons, and discussion forums to improve understanding and engagement.
